# Astrologer Appointment App

A simple PHP application that lets users browse astrologer services and book appointments. An admin page lists all scheduled appointments.

## Setup

1. Start the built-in PHP server:
   ```bash
   php -S localhost:8000
   ```
2. Open `http://localhost:8000/index.php` in your browser.

## Features
- Browse a list of services.
- Book an appointment for a service.
- Admin view of all appointments at `/admin.php`.
